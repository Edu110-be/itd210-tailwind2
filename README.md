- Honda Legacy – Capstone Project

Name: Yididiya Meresa 
Milestone: Gallery Page (Milestone 3)

- Project Overview

This project is a multi-page website based on Honda’s automotive history. It includes pages such as Home, About, Gallery, Media, and Contact, each focusing on a different part of the theme.

- What’s New in This Milestone

In this milestone, I built and improved the Gallery page. It includes a responsive grid layout with nine images of Honda vehicles. Each image has hover effects, captions, and alt text for accessibility. I also added a lightbox feature so users can click images to view them in a larger format.

The page includes keyboard focus support, reduced motion settings, and consistent navigation across all pages.

- Technologies Used

HTML, CSS, JavaScript, and GitHub Pages.
