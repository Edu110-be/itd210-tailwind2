 Honda Legacy - Capstone Project

Name: Yididiya Meresa  



Project Description -

This project is a multi-page website designed to showcase the different cars of Honda while also telling the story of the brand’s past, present, and future. It highlights Honda’s automotive legacy through innovation, design, and performance across different eras.

The website allows users to explore Honda’s history, view image galleries, watch media content, and interact through a contact page. Each section is designed to create a smooth and engaging experience that reflects Honda’s evolution in the automotive world.



Live URL
https://edu110-be.github.io/itd210-tailwind2/



Technologies Used -

HTML, CSS, JavaScript, and Tailwind


Features - 

- Flexbox 
- Card-based homepage structure
- Hover effects with smooth transitions
- Sticky navigation bar
- Hero section with image and text
- Accessible images with alt text
- Keyboard focus states 
- Reduced motion support for animations
- Responsive design for mobile and desktop
- External JavaScript file linked (script.js)


What I Learned -

Through this project, I learned how to build a complete multi-page website using HTML, CSS, and JavaScript. I improved my understanding of layout design, responsiveness, and how to structure a clean and user-friendly interface.

I also learned how to:

- Use Flexbox to build responsive layouts
- Create interactive hover animations
- Improve accessibility with alt text and focus states
- Organize navigation across multiple pages
- Connect external JavaScript files for scalability
- Design a consistent theme across all pages

  
Strongest Part of the Site -

The strongest part of my site is the homepage layout with interactive cards and navigation. It is simple, visually clear, and helps users easily move between pages. The hover effects and transitions also make the site feel more modern and engaging.


Responsive Design & Interactivity -

I learned that responsive design is not just shrinking elements but reorganizing layouts using Flexbox and media queries so the site works on all screen sizes.


Feedback That Helped Most - 

The most helpful feedback I received was to comment my code and organize it into clear sections. This made my code much easier to read, debug, and maintain. It also helped me understand my own structure better when reviewing it later.


Future Improvements -

If I had more time, I would add more advanced JavaScript features like a gallery slider, and optimize performance by compressing images.
