Honda Legacy – Capstone Project

Name: Yididiya Meresa  
Milestone: Media Page (Milestone 4)


- Project Overview

This project is a multi-page website based on Honda’s automotive history. It includes pages such as Home, About, Gallery, Media, and Contact, each focusing on a different part of the theme.


- What’s New in This Milestone

In this milestone, I created the Media page to add an interactive experience to the website. The page includes a YouTube video that showcases Honda engineering and innovation, along with an audio file named engine.mp3 that represents engine sounds.

I also added interactive tabs so users can switch between video and audio content. The page includes a CSS animation that represents motion and performance, as well as a JavaScript play/pause button for controlling the audio.

To improve accessibility, I included transcripts for both the video and audio content, along with keyboard-friendly controls and reduced motion support.


- Technologies Used

HTML, CSS, JavaScript, and GitHub Pages.
